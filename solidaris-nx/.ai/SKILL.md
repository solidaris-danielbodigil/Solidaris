## AI Skills Overview

This document outlines the skills and capabilities of AI agents in the UI design context for the Solidaris NX project, specifically for the applications iSHARE and iCRM.

### General Skills

- **Understanding Design Systems**: Ability to comprehend and implement design systems, specifically the Plectrum design system, ensuring consistency across applications.
- **Component Design**: Proficient in designing reusable UI components following BEMIT naming conventions and ITCSS structure.
- **SCSS Proficiency**: Skilled in writing and organizing SCSS using ITCSS methodology, including variables, mixins, and utility classes.

### Application-Specific Skills

#### iSHARE Application

- **Routing Configuration**: Capable of defining and managing routing configurations for the iSHARE application.
- **State Management**: Understanding of state management techniques suitable for the iSHARE application context.
- **Asset Management**: Knowledgeable in managing static assets and integrating them into the application.

#### iCRM Application

- **API Integration**: Proficient in integrating APIs and handling data fetching for the iCRM application.
- **User Experience Design**: Ability to design user-friendly interfaces that enhance the user experience in the iCRM application.

### Collaboration Skills

- **Storybook Integration**: Familiar with using Storybook for developing and showcasing UI components in isolation.
- **Version Control**: Understanding of Git workflows for collaborative development and continuous integration practices.

### Continuous Learning

- **Staying Updated**: Commitment to staying updated with the latest trends in Angular, PrimeNG, and UI design best practices.
- **Feedback Incorporation**: Ability to incorporate feedback from design reviews and user testing to improve UI components and overall application design.
